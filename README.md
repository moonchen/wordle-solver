# wordle-solver

## Command Line

`python3 wordle-solve.py <positionals> <known letters>  <known wrong letters>`

### Examples of each input:


|positionals|known letters|known wrong letters|
|-----------|-------------|-------------------|
|`_____`    |`abc`        |`def`              |
|`ab__t`    |`ou`         |`spqnx`            |
|`____e`    |`_`          |`ocpaonw`          |
|`q___n`    |`u`          |`_`                |
